### 代码说明：

**CipherFour.cpp**

命令行输入make 可以直接运行，打印出task2，task3，task4的结果

或者命令行g++ CipherFour.cpp -O2 编译成二进制文件运行

**task5.py**

执行task5()函数可以根据一轮最优差分路径，构造R轮加密的最优差分，R可手动修改

输入s_num并执行task5_new()可以寻找第s_num个S盒的最优差分特征
